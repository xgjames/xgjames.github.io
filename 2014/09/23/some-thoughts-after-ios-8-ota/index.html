Some thoughts after iOS 8 OTA

23 September 2014

It’s like a custom that upgrading iOS OTA (Over the Air). I’m familiar with iCloud backup and restore, being aware of that iCloud network could never be fully functional using PRC Internet. You have a data cloud and you use it. Lightening is for charging only. Seems straight.
Still, the requirement of 5.8GB storage for installing iOS 8 on my iPhone 5s is kinda surprising. Good time to ditch those apps you seldom use and get more storage released. Of course the best option is to get a 64GB or 128GB device next time.
More excitement comes from Xcode 6.0.1, since I’ve been working as a full-time developer for over 3 months. It’s never too late to learn, especially to learn Swift. Some colleagues have dived into the new programming language since the beta version was released.
